# RN Command Panel Renoise Plugin

This is a xrnx plugin for Renoise. 
I put this together for my own use. It is a mix of features from some of my favorite tools by other plugin developers. I just wanted to combine stuff into one plugin and make it all more efficient on my system by reducing plugins I need to load separately when starting up Renoise. 

It can:

- add effects
- change volume, pan, delay on one or multiple selected rows or a whole pattern
- change effect amounts
- make quick selections of tracks and sections
- ramp up and down column
- ramp up and down retriggers
- randomize effects parameters with probabilities

More things will be added into the panels if I need something else but for now I want to keep it simple.
